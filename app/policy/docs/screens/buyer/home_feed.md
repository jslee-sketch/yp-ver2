# 홈 피드 화면 (Home Feed)

> 최종 수정: 2026-02-28  
> 우선순위: ★★★★☆  
> 역할: "역핑에 들어오면 가장 먼저 보는 화면" — 라이브 딜이 흐르는 느낌

---

## 1. 진입 경로

- 앱 실행 시 기본 화면
- 하단 탭 바 "홈" 탭
- 로그인 후 리다이렉트

---

## 2. 사용하는 API

| API | 용도 | 호출 시점 |
|-----|------|----------|
| GET /deals/?status=open&sort=latest&size=20 | 진행 중 딜 목록 (최신순) | 화면 진입 |
| GET /deals/?status=open&sort=popular&size=10 | 인기 딜 (참여자 많은 순) | 화면 진입 |
| GET /deals/?status=open&sort=closing_soon&size=5 | 마감 임박 딜 | 화면 진입 |
| GET /spectator/rankings?period=monthly&limit=5 | 이달의 관전자 랭킹 | 화면 진입 |
| GET /dashboard/buyer/{id} | 내 요약 (포인트, 진행 중 딜) | 로그인 시 |
| POST /deals/ | 새 딜 생성 | 딜 생성 버튼 |

## 3. 화면 구성 (위→아래 순서)

### 3-1. 상단 바

```
역핑                          [🔔 3]  [⚙️]
```

- 좌측: 역핑 로고/워드마크
- 우측: 알림 (읽지 않은 수), 설정
- 배경: var(--bg-primary), 스크롤 시 blur 효과

### 3-2. 내 요약 카드 (로그인 시)

```
┌─────────────────────────────────────────────┐
│  안녕하세요, 이정상님! 👋                     │
│                                               │
│  💰 포인트: 2,350P    🎯 진행 중 딜: 3개      │
│  👀 관전 중: 5개       🏆 이달 랭킹: 12위      │
└─────────────────────────────────────────────┘
```

데이터 매핑:
- 포인트: dashboard.points_balance
- 진행 중 딜: dashboard.active_deals_count
- 관전 중: dashboard.watching_deals_count
- 랭킹: spectator_rankings에서 내 순위

비로그인 시:
```
┌─────────────────────────────────────────────┐
│  역핑에서 원하는 가격에 사세요 🎯              │
│  [ 시작하기 ]                                 │
└─────────────────────────────────────────────┘
```

### 3-3. 실시간 피드 띠 (Live Ticker)

```
───────────────────────────────────────────
⚡ 방금: 에어팟 프로 2에 89,000원 오퍼 도착!
⚡ 2분 전: 갤럭시 S25 딜에 관전자 5명 참여
⚡ 5분 전: 나이키 에어맥스 딜 성사! 115,000원
───────────────────────────────────────────
```

- 수평 스크롤 또는 자동 롤링 (3초 간격)
- 최근 10개 이벤트
- 탭하면 해당 딜 상세로 이동
- 데이터: activity_log 최신 항목 또는 SSE 실시간 이벤트

### 3-4. 마감 임박 딜 (가로 스크롤)

```
⏰ 마감 임박!

┌──────────┐ ┌──────────┐ ┌──────────┐
│ 에어팟    │ │ 갤럭시    │ │ 다이슨    │
│ 프로 2   │ │ S25      │ │ 에어랩    │
│          │ │          │ │          │
│ 89,000원  │ │ 1.2M     │ │ 550K     │
│ ⏰ 2시간  │ │ ⏰ 5시간  │ │ ⏰ 8시간  │
│ 👀 87    │ │ 👀 142   │ │ 👀 56    │
└──────────┘ └──────────┘ └──────────┘
             ← 가로 스크롤 →
```

카드 데이터:
- 상품명: deal.product_name (2줄 말줄임)
- 최저 오퍼: offers[0].price
- 남은 시간: deadline_at - now (빨간색 강조)
- 관전자 수: spectator_count

마감 임박 기준: 24시간 이내 마감 예정
카드 너비: 140px, 높이: 160px
배경: var(--bg-secondary), border: accent-red

### 3-5. 인기 딜 (세로 카드 피드)

```
🔥 지금 뜨는 딜

┌─────────────────────────────────────────────┐
│  ● LIVE  에어팟 프로 2세대 (USB-C)           │
│  Apple                                       │
│                                               │
│  📊 103,900원 → ⚡ 89,000원  (-14%)          │
│                                               │
│  참여 12명  ·  오퍼 4개  ·  👀 87명 관전       │
│  ████████████████░░░  목표 93%               │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│  ● LIVE  갤럭시 S25 울트라 512GB              │
│  Samsung                                     │
│                                               │
│  📊 1,578,830원 → ⚡ 1,450,000원  (-8%)      │
│                                               │
│  참여 28명  ·  오퍼 7개  ·  👀 142명 관전      │
│  █████████████████████  목표 달성!            │
└─────────────────────────────────────────────┘
```

카드 데이터:
- 상태 배지: LIVE (녹색) / 마감 (회색)
- 상품명: deal.product_name
- 브랜드: deal.brand
- 시장가: deal.anchor_price (취소선)
- 최저 오퍼: offers[0].price + 절약 퍼센트
- 참여자/오퍼/관전자 수
- 목표 달성률 프로그레스 바

카드 탭 → 딜 상세 화면 이동

정렬 옵션 (카드 상단 탭):
- 최신순 (기본)
- 인기순 (참여자 + 관전자 합산)
- 절약률순 (시장가 대비 절약 % 높은 순)
- 마감순

### 3-6. 관전자 랭킹 미니 위젯

```
🏆 이달의 관전왕

 1위  spectator_kim   적중 12회  +240P
 2위  eagle_eye_99    적중 10회  +180P
 3위  price_hunter    적중 9회   +160P

         [ 전체 랭킹 보기 ]
```

데이터: GET /spectator/rankings?period=monthly&limit=3
- 이름, 적중 횟수, 획득 포인트
- [전체 랭킹 보기] → 관전자 랭킹 전체 화면

### 3-7. 딜 생성 CTA (하단 고정)

```
┌─────────────────────────────────────────────┐
│     원하는 가격에 사고 싶은 게 있나요?         │
│                                               │
│          [ ➕ 새 딜 만들기 ]                   │
└─────────────────────────────────────────────┘
```

- 스크롤 최하단에 도달 시 표시
- 또는 하단 탭 바의 ➕ 버튼과 동일 기능

## 4. 새로고침/페이지네이션

| 동작 | 구현 |
|------|------|
| 당겨서 새로고침 (Pull to Refresh) | 전체 데이터 리패치 |
| 무한 스크롤 | 인기 딜 섹션, 20개씩 커서 기반 페이지네이션 |
| 자동 갱신 | 실시간 피드 띠: 30초 간격 폴링 |

---

## 5. 검색 진입

상단 바 아래에 검색 바 (탭하면 검색 화면으로 이동):

```
🔍 어떤 제품을 찾고 계세요?
```

- 탭 → 검색 화면 (별도)으로 전환
- 최근 검색어, 인기 검색어 표시
- 검색 결과: GET /deals/?keyword={query}&status=open

---

## 6. 핑퐁이 플로팅 버튼

```
                                    ┌─────┐
                                    │ 🤖  │
                                    └─────┘
                          (화면 우하단, 탭 바 위)
```

- 탭 → 핑퐁이 채팅 바텀 시트 오픈
- 홈에서의 핑퐁이: 일반 질문 모드 (딜 추천, 포인트 확인, 정책 질문 등)
- 딜 상세에서의 핑퐁이: 해당 딜 맥락 모드

---

## 7. 에러/빈 상태

| 상태 | 표시 |
|------|------|
| 로딩 중 | 스켈레톤 UI (카드 3개 플레이스홀더 + 실시간 피드 빈 줄) |
| 딜 0개 | "아직 진행 중인 딜이 없어요. 첫 딜을 만들어보세요!" + CTA |
| 네트워크 에러 | "연결이 불안정해요" + 재시도 |
| 비로그인 | 내 요약 대신 가입 유도 배너, 나머지 정상 표시 |

---

## 8. 핑퐁이 홈 안내 문구

| 상황 | 문구 |
|------|------|
| 첫 방문 | "반가워요! 역핑은 원하는 가격에 사는 새로운 방법이에요. 궁금한 거 물어보세요! 🤖" |
| 재방문 (진행 딜 있음) | "진행 중인 딜이 {N}개 있어요. 새 오퍼가 들어왔는지 확인해볼까요?" |
| 재방문 (관전 중) | "관전 중인 딜 {N}개에서 오퍼가 업데이트됐어요!" |
| 예측 적중 | "🎉 축하해요! 어제 {딜명} 예측이 적중했어요! +{N}P 적립됐어요." |

---

## 9. 딜 카드 스와이프 액션

딜 카드를 좌/우 스와이프:
- 우측 스와이프: "관전하기" (딜 관전 등록 + 예측 화면)
- 좌측 스와이프: "관심없음" (피드에서 숨김)

---

## 10. 카테고리 필터 (상단 가로 스크롤)

```
[전체] [전자기기] [패션] [식품] [생활용품] [뷰티] [스포츠] ...
```

- 가로 스크롤 칩(chip) 형태
- 탭 시 해당 카테고리 딜만 필터
- "전체"가 기본 선택
- 데이터: GET /deals/?category={category}
