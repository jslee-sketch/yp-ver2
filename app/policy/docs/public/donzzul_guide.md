# 돈쭐 가이드 (Donzzul Guide)

## 돈쭐이란?
돈쭐은 착한 가게를 응원하는 상품권 시스템입니다. "돈으로 혼쭐내다"의 줄임말로, 선행을 베푸는 가게에 상품권을 사서 응원하는 문화입니다.

## 용어 정리
- **돈쭐 히어로**: 착한 가게를 발굴하고 추천하는 사용자
- **돈쭐 딜**: 특정 가게를 응원하기 위한 상품권 판매 캠페인
- **돈쭐 상품권**: 해당 가게에서 사용할 수 있는 디지털 상품권
- **응원방**: 딜별 채팅방에서 가게에 응원 메시지를 남길 수 있음

## 이용 흐름

### 1. 가게 추천 (돈쭐 히어로)
- 착한 가게를 발견하면 `/donzzul/hero/recommend`에서 가게 정보를 등록
- 필수 정보: 가게명, 주소, 전화번호, 사장님 정보, 계좌 정보, 사연 (최소 50자)
- 유튜브 영상 URL을 추가할 수 있음
- 관리자 검증 후 승인되면 돈쭐 딜이 자동 생성됨
- 히어로 포인트 500점 적립 (가게당)

### 2. 상품권 구매
- 금액 선택: 1만원, 2만원, 5만원
- 응원 한마디를 남길 수 있음 (선택, 최대 200자)
- 결제 후 상품권 코드(DONZZUL-XXXX-XXXX)와 4자리 PIN 발급
- PIN은 발급 시 한 번만 표시되므로 반드시 기억할 것
- 유효기간: 90일

### 3. 상품권 사용
- 가게 방문 후 `/donzzul/vouchers`에서 "사용하기" 클릭
- 사장님이 4자리 가게 비밀번호를 입력하면 사용 확정
- PIN 5회 오류 시 30분간 잠금

### 4. 만료 및 기부
- 유효기간 내 미사용 상품권은 자동으로 가게에 기부됨 (DONATED)
- 만료 14일/7일/1일 전에 알림 발송
- 역핑 수수료: 0원 (전액 가게에 전달)

### 5. 정산
- 사용(USED) + 만료 기부(DONATED) 상품권 합산
- 관리자가 정산 생성 → 승인 → 지급 순서로 처리
- 수수료 0원 → 전액 가게 계좌로 입금

## 히어로 레벨
- 새싹: 가게 1곳 이상
- 착한: 가게 3곳 이상
- 슈퍼: 가게 10곳 이상
- 레전드: 가게 30곳 이상

## 응원 채팅방
- 각 딜마다 응원 채팅방이 있음
- 닉네임 설정 후 실시간 채팅 가능
- 가게에 응원 메시지를 남길 수 있음
- 5초 간격 자동 새로고침

## 자주 묻는 질문

Q: 상품권을 환불할 수 있나요?
A: 돈쭐 상품권은 착한 가게 응원 목적이므로 일반 환불이 불가합니다. 단, 가게가 폐업하는 등 특수한 경우 관리자에게 문의해주세요.

Q: PIN을 잊어버렸어요.
A: 안타깝지만 PIN은 발급 시 한 번만 표시됩니다. 분실 시 관리자에게 문의해주세요.

Q: 상품권이 만료되면 어떻게 되나요?
A: 유효기간이 지난 상품권은 자동으로 해당 가게에 기부됩니다. 가게 사장님께 전액 전달됩니다.

Q: 수수료가 정말 0원인가요?
A: 네, 역핑은 돈쭐 서비스에서 수수료를 받지 않습니다. 상품권 금액 전액이 가게에 전달됩니다.

Q: 여러 가게에 상품권을 살 수 있나요?
A: 네, 오픈된 모든 딜에서 상품권을 구매할 수 있습니다.
