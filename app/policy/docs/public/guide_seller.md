# 판매자 완전 가이드 (guide_seller)

## 1. 판매자 등록
1. 회원가입 → 판매자 전환 신청
2. 사업자 정보 입력: 상호명, 사업자번호, 대표자명
3. 서류 업로드: 사업자등록증, 통신판매업신고증(선택), 통장사본
4. 관리자 승인 대기 (보통 1-2일)
5. 승인 완료 → 오퍼 제출 가능

---

## 2. 오퍼 제출
구매자가 만든 딜에 판매 조건을 제안합니다.

### 오퍼 작성 항목
| 항목 | 설명 |
|------|------|
| 단가 | 개당 판매 가격 |
| 수량 | 공급 가능 수량 |
| 배송비 | 배송비 (무료배송 시 0) |
| 배송 방식 | 택배/직거래/무료배송 |
| 환불 조건 | 환불 가능 여부 및 상세 조건 |
| 유효기간 | 오퍼 만료 기한 (기본 48시간) |

### 오퍼 전략 팁
- 공동구매 참여자가 많을수록 자동 가격 인하 적용
- 배송비 무료 오퍼가 선택률이 높음
- 환불 조건이 유연할수록 구매자 신뢰 상승

---

## 3. 주문 처리
구매자가 오퍼를 선택하고 결제하면:
1. **판매자 대시보드**에서 새 주문 확인
2. 상품 준비 & 포장
3. **배송 처리**: 택배사 + 운송장번호 입력
4. 구매자 도착 확인 대기

### 배송 처리 주의사항
- 결제 확인 후 **48시간 이내** 배송 처리 권장
- 3일 초과 시 관리자 모니터링 대상
- 운송장번호 오입력 시 수정 가능

---

## 4. 정산
```
HOLD (쿨링 7일) → READY → APPROVED (관리자 승인) → PAID (입금)
```

### 정산 금액 계산
```
정산액 = 결제금액 - PG수수료(3.3%) - 플랫폼수수료(레벨별)
```

### 플랫폼 수수료 (레벨별)
| 레벨 | 거래 건수 | 최소 평점 | 수수료율 |
|------|----------|----------|---------|
| Lv.6 (신규) | 0-20 | - | 3.5% |
| Lv.5 | 21-40 | 4.0 | 3.0% |
| Lv.4 | 41-60 | 4.0 | 2.8% |
| Lv.3 | 61-100 | 4.0 | 2.7% |
| Lv.2 | 101+ | 4.0 | 2.5% |
| Lv.1 | 101+ | 4.5 | 2.0% |

> 분쟁 중인 거래는 정산이 자동 보류됩니다.

---

## 5. 반품/환불 처리
구매자가 환불 요청 시:
1. **환불 관리** 메뉴에서 요청 확인
2. 사유 확인 및 승인/거부 결정
3. 반품 시: 상품 회수 후 환불 처리
4. 정산에서 환불 금액 차감

### 환불 유형
- **refund**: 단순 환불 (배송 전)
- **return**: 반품 (배송 후, 상품 회수 필요)
- **exchange**: 교환 (새 상품 재배송)

---

## 6. 분쟁 대응
구매자가 분쟁을 접수하면:
1. 알림 수신 → **정산 관리**에서 확인
2. 관리자가 양측 의견 확인
3. 증빙 자료 제출 (배송 증빙, 상품 사진 등)
4. 관리자 판정: 환불/부분환불/기각
5. 결과에 따라 정산 조정

> 분쟁이 자주 발생하면 레벨 하락 및 계정 제한 가능

---

## 7. 리뷰 관리
- 구매자 도착 확인 후 30일 이내 리뷰 작성 가능
- 판매자는 리뷰에 **답글** 작성 가능
- 평점이 레벨 산정에 반영됨

### 좋은 평점을 위한 팁
- 빠른 배송 (48시간 이내)
- 정확한 상품 설명
- 친절한 문의 대응
- 안전한 포장

---

## 8. 배송 정책 설정
판매자별 배송 정책을 미리 설정할 수 있습니다:
- 기본 배송비
- 무료 배송 기준 금액
- 제주/도서산간 추가 배송비
- 교환/반품 배송비

**판매자 메뉴** → **배송 정책**에서 설정

---

## 9. 통계 & 분석
판매자 대시보드에서 확인 가능:
- 총 매출 / 기간별 매출 추이
- 오퍼 채택률
- 평균 배송 소요일
- 리뷰 평점 추이
- 환불률

---

## 10. 자주 묻는 질문
- **Q: 오퍼 수정 가능?** → 구매자 결제 전까지 수정/철회 가능
- **Q: 정산 언제?** → 쿨링(7일) + 관리자 승인 후 지급 (약 30일)
- **Q: 분쟁 보류 해제?** → 분쟁 종료 후 3일 후 정산 재개
- **Q: 수수료 인하?** → 거래 실적+평점 쌓아서 레벨업하면 자동 인하
- **Q: 액추에이터?** → 판매자 모집/관리하는 중간자, 커미션 별도 정산
