# 액추에이터 가이드

## 액추에이터란?
액추에이터(Actuator)는 역핑 플랫폼에서 판매자를 모집하고 관리하는 중간자 역할입니다.
판매자와 플랫폼을 연결하여 거래 활성화를 돕고, 거래 성사에 따라 커미션을 받습니다.

---

## 회원가입 / 로그인
이메일 또는 소셜(카카오/네이버/구글) 로그인으로 가입합니다.
회원가입 시 "액추에이터" 역할을 선택합니다.
소셜 로그인 시에도 역할 선택 → 닉네임 설정 → 완료 후 자동 로그인됩니다.

---

## 주요 기능

### 판매자 모집
- 추천 코드를 생성하여 판매자에게 공유
- 판매자가 추천 코드로 가입하면 자동 연결
- 연결된 판매자 목록 확인
> 자주 묻는 표현: 판매자 모집, 추천 코드, 판매자 연결, 여러 판매자 관리

### 딜 탐색 & 오퍼 지원
- 현재 열린 딜 목록을 탐색하여 판매자에게 적합한 딜 안내
- 판매자의 오퍼 제출을 도와줄 수 있음 (직접 제출은 판매자만 가능)
- 딜 매칭: 판매자의 강점 카테고리/브랜드에 맞는 딜 추천
> 자주 묻는 표현: 딜 탐색, 오퍼 제출 도움, 오퍼 지원, 딜 추천

### 배송 관리
- 연결된 판매자의 배송 현황 모니터링
- 배송 지연 시 판매자에게 알림/독촉
- 직접 배송 처리는 판매자만 가능
> 자주 묻는 표현: 배송 관리, 배송 모니터링, 배송 현황

### 리뷰 관리
- 연결된 판매자의 리뷰/평점 현황 확인
- 리뷰 답글 작성은 판매자만 가능

## 커미션 구조
액추에이터 커미션은 연결된 판매자의 거래 성사 금액 기반으로 레벨별 차등 적용됩니다.

| 레벨 | 거래 건수 | 커미션율 |
|------|----------|---------|
| Lv.6 (신규) | 0-20 | 0.5% |
| Lv.5 | 21-40 | 0.2% |
| Lv.4 | 41-60 | 0.1% |
| Lv.3-1 | 61+ | 0% |

> 커미션은 판매자 정산과 별도로 지급됩니다.
> 자주 묻는 표현: 커미션 구조, 커미션 얼마, 수수료 구조, 커미션 레벨, 커미션율

---

## 위탁계약서

커미션 정산을 받으려면 **위탁계약서 동의**가 필요합니다.

- 경로: 사이드바 → 위탁계약서
- 계약서 전문(14조)을 끝까지 스크롤한 후 3개 항목에 동의
- 한 번 동의하면 재동의 불필요 (버전 변경 시 재동의 안내)

> 자주 묻는 표현: 위탁계약서, 계약서 동의, 계약 어디서, 계약서 내용

---

## 정산

### 정산 흐름
- 커미션은 연결된 판매자의 거래가 구매확정된 후 정산됩니다.
- 정산 흐름: HOLD (쿨링 7일) → READY → APPROVED → PAID
- 판매자 정산과 동일한 파이프라인을 따릅니다.

### 개인 액추에이터 (원천징수)
- 소득세 3% + 지방소득세 0.3% = **총 3.3% 원천징수** 후 지급
- 예시: 커미션 100,000원 → 원천징수 3,300원 → 실수령 96,700원
- 원천징수 영수증은 매년 2월 말 교부

### 사업자 액추에이터 (세금계산서)
- 사업자 등록된 액추에이터는 원천징수 대신 세금계산서 발행
- 정산 승인 시 세금계산서 자동 생성 → 확인 → 발행
- 세금계산서 확인: 사이드바 → 세금계산서 (판매자와 동일 플로우)

### 최소 지급액
- 10,000원 미만 시 다음 정산 주기로 이월

> 자주 묻는 표현: 정산 언제, 정산 방법, 커미션 정산, 정산 어떻게 받아, 원천징수, 세금, 3.3%, 세금계산서

## 자주 묻는 질문
- **Q: 액추에이터 역할이 뭐야?** → 판매자를 모집/관리하는 중간자, 거래 성사 시 커미션 수령
- **Q: 커미션 얼마야?** → Lv.6 기준 0.5%, 레벨업 시 감소 (Lv.3 이상 0%)
- **Q: 여러 판매자 관리 가능?** → 가능, 추천 코드로 여러 판매자 연결
- **Q: 정산은 어떻게 받아?** → 판매자 거래 구매확정 후 쿨링(7일) + 승인 후 지급
- **Q: 원천징수 얼마야?** → 개인 액추에이터: 소득세 3% + 지방소득세 0.3% = 3.3%
- **Q: 사업자인데 세금계산서 발행 가능해?** → 사업자 등록 시 세금계산서 발행 (원천징수 아님)
- **Q: 위탁계약서 어디서 봐?** → 사이드바 → 위탁계약서 → 끝까지 읽고 동의
- **Q: 계약서 안 하면?** → 계약 동의 전까지 커미션 정산 불가
- **Q: 오퍼 제출 도와줄 수 있어?** → 딜 안내 가능, 직접 오퍼 제출은 판매자만
- **Q: 배송 관리도 해야 해?** → 모니터링만 가능, 직접 배송 처리는 판매자
