# 액추에이터 가이드

## 액추에이터란?
액추에이터(Actuator)는 역핑 플랫폼에서 판매자를 모집하고 관리하는 중간자 역할입니다.
판매자와 플랫폼을 연결하여 거래 활성화를 돕고, 거래 성사에 따라 커미션을 받습니다.

---

## 회원가입 / 로그인
이메일 또는 소셜(카카오/네이버/구글) 로그인으로 가입합니다.
회원가입 시 "액추에이터" 역할을 선택합니다.
소셜 로그인 시에도 역할 선택 → 닉네임 설정 → 완료 후 자동 로그인됩니다.

---

## 주요 기능

### 판매자 모집
- **추천코드**를 생성하여 판매자에게 공유
- 판매자가 추천코드로 가입하면 자동 연결
- 연결된 판매자 목록 확인
- 추천코드 확인: /actuator/referral
> 자주 묻는 표현: 판매자 모집, 추천 코드, 추천코드, 판매자 연결, 여러 판매자 관리, 추천코드 확인

### 딜 탐색 & 오퍼 지원
- 현재 열린 딜 목록을 탐색하여 판매자에게 적합한 딜 안내
- 판매자의 오퍼 제출을 도와줄 수 있음 (직접 제출은 판매자만 가능)
- 딜 매칭: 판매자의 강점 카테고리/브랜드에 맞는 딜 추천
> 자주 묻는 표현: 딜 탐색, 오퍼 제출 도움, 오퍼 지원, 딜 추천

### 배송 관리
- 연결된 판매자의 배송 현황 모니터링
- 배송 지연 시 판매자에게 알림/독촉
- 직접 배송 처리는 판매자만 가능
> 자주 묻는 표현: 배송 관리, 배송 모니터링, 배송 현황

### 리뷰 관리
- 연결된 판매자의 리뷰/평점 현황 확인
- 리뷰 답글 작성은 판매자만 가능

## 커미션 구조
액추에이터 커미션은 연결된 판매자의 거래 성사 금액 기반으로 레벨별 차등 적용됩니다.

| 레벨 | 거래 건수 | 커미션율 |
|------|----------|---------|
| Lv.6 (신규) | 0-20 | 0.5% |
| Lv.5 | 21-40 | 0.2% |
| Lv.4 | 41-60 | 0.1% |
| Lv.3-1 | 61+ | 0% |

> 커미션은 판매자 정산과 별도로 지급됩니다.
> 자주 묻는 표현: 커미션 구조, 커미션 얼마, 수수료 구조, 커미션 레벨, 커미션율

## 위탁계약서

커미션 정산을 받으려면 **위탁계약서 동의**가 필요합니다.

- 경로: /actuator/contract (사이드바 → 위탁계약서)
- 계약서 전문(14조)을 끝까지 스크롤한 후 3개 항목에 동의
  - 계약 내용 동의 / 개인정보 수집 동의 / 비밀유지 동의
- 한 번 동의하면 재동의 불필요 (버전 변경 시 재동의 안내)
- 계약 기간: 1년 (자동 연장)
- 해지: 30일 전 서면 통지

> 자주 묻는 표현: 위탁계약서, 계약서 동의, 계약 어디서, 계약서 내용, 계약 기간, 해지

## 사업자 vs 개인

### 사업자 액추에이터
- 세금계산서 발행 (지급일 7일 전)
- 사업자등록증 업로드 → OCR 자동 파싱 → 정보 확인
- 세금계산서 이메일 입력 필요
- /actuator/business-info에서 사업자 정보 변경 가능
> 자주 묻는 표현: 사업자 액츄에이터, 사업자 세금계산서, 사업자등록증 업로드, 사업자 정보

### 개인 액추에이터
- **3.3% 원천징수** 후 계좌 입금 (소득세 3% + 지방소득세 0.3%)
- 예시: 커미션 100,000원 → 원천징수 3,300원 → 실수령 96,700원
- 매년 5월 종합소득세 신고 필요
- 원천징수영수증 PDF 다운로드 가능 (/actuator/settlements)
> 자주 묻는 표현: 개인 액츄에이터, 원천징수, 3.3%, 종합소득세, 원천징수영수증, PDF 다운로드

## 커미션 정산
모집한 판매자의 거래 성사 시 커미션 발생.
- 정산 흐름: 구매확정 → **HOLD(쿨링 7일)** → READY → **APPROVED(관리자 승인)** → PAID(지급)
- 판매자 정산과 동일한 파이프라인
- 정산 내역 확인: /actuator/settlements
- **정산내역서 PDF 다운로드** 가능

### 최소 지급액
- **10,000원** 미만 시 다음 정산 주기로 이월

> 자주 묻는 표현: 정산 언제, 정산 방법, 커미션 정산, 정산 어떻게 받아, 최소 지급액, 10000원, 정산내역서, PDF

## 알림
경로: /notifications
커미션 발생, 정산 준비 완료, 세금계산서 발행 등 알림을 확인합니다.
**푸시 알림(FCM)**: 브라우저 알림을 허용하면 실시간 알림을 받을 수 있습니다.
> 자주 묻는 표현: 알림, 푸시 알림, FCM, 알림 설정

### 알림 설정
경로: /settings/notifications
13개 액추에이터 이벤트(커미션/정산/판매자/계약 등)별 앱/푸시/이메일 알림을 설정합니다.
> 자주 묻는 표현: 알림 설정, 알림 끄기, 푸시 설정

### 관심 상품
경로: /settings/interests
관심 카테고리/제품/모델을 등록하면 관련 딜이 생성될 때 자동 알림을 받습니다.
최대 10개까지 등록 가능. 회원가입 시에도 등록 가능 (선택사항).
> 자주 묻는 표현: 관심 상품, 관심 등록, 딜 알림, 매칭 알림

## 판매자 해지 프로세스
연결된 판매자와의 관계를 해지할 수 있습니다.

### 해지 사유 (5가지)
- 신뢰 문제 (소통 불량, 약속 미이행)
- 성과 부진 (판매/중개 실적 미달)
- 정책 위반 (허위 정보, 부당 행위)
- 상호 합의에 의한 해지
- 개인 사정 (사업 방향 변경 등)

### 해지 절차
1. 해지 약관 동의 → 해지 신청
2. **유예 기간 7일**: 이 기간 내 철회 가능
3. 유예 만료 → 해지 확정 (CONFIRMED)
4. **쿨다운 30일**: 동일 판매자와 재연결 불가

> 자주 묻는 표현: 판매자 해지, 해지 신청, 해지 철회, 유예 기간, 쿨다운, 재연결

## 자주 묻는 질문
- **Q: 액추에이터 역할이 뭐야?** → 판매자를 모집/관리하는 중간자, 거래 성사 시 커미션 수령
- **Q: 커미션 얼마야?** → Lv.6 기준 0.5%, 레벨업 시 감소 (Lv.3 이상 0%)
- **Q: 여러 판매자 관리 가능?** → 가능, 추천 코드로 여러 판매자 연결
- **Q: 정산은 어떻게 받아?** → 판매자 거래 구매확정 후 쿨링(7일) + 승인 후 지급
- **Q: 원천징수 얼마야?** → 개인 액추에이터: 소득세 3% + 지방소득세 0.3% = 3.3%
- **Q: 사업자인데 세금계산서 발행 가능해?** → 사업자 등록 시 세금계산서 발행 (원천징수 아님)
- **Q: 위탁계약서 어디서 봐?** → /actuator/contract → 끝까지 읽고 3개 동의
- **Q: 계약서 안 하면?** → 계약 동의 전까지 커미션 정산 불가
- **Q: 오퍼 제출 도와줄 수 있어?** → 딜 안내 가능, 직접 오퍼 제출은 판매자만
- **Q: 배송 관리도 해야 해?** → 모니터링만 가능, 직접 배송 처리는 판매자
- **Q: 최소 지급액?** → 10,000원 미만 시 다음 정산으로 이월
- **Q: 계좌 등록 어떻게?** → /actuator/bank-info에서 본인 명의 계좌 등록
- **Q: 원천징수영수증 어디서 받아?** → /actuator/settlements에서 PDF 다운로드
- **Q: 정산내역서 받을 수 있어?** → /actuator/settlements에서 PDF 다운로드
- **Q: 푸시 알림 어떻게 받아?** → 브라우저 알림 허용하면 커미션/정산/세금계산서 알림 수신
