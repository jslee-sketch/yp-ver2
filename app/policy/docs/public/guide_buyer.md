# 구매자 완전 가이드 (guide_buyer)

## 1. 역핑이란?
역핑(Yeokping)은 **역경매 공동구매 플랫폼**입니다.
구매자가 원하는 상품과 희망 가격을 먼저 제시하면, 판매자들이 경쟁적으로 오퍼를 제출합니다.

### 기존 쇼핑과 차이점
| 기존 쇼핑 | 역핑 |
|-----------|------|
| 판매자가 가격 결정 | 구매자가 희망가 제시 |
| 1:1 거래 | 공동구매로 가격 하락 |
| 가격 비교 어려움 | 오퍼 비교로 최적 선택 |

---

## 2. 딜 만들기
1. 홈 → **[+ 딜 만들기]** 탭
2. 상품명, 카테고리, 희망 가격, 수량, 옵션 입력
3. 마감 기한 설정 (기본 72시간)
4. 딜 생성 완료 → 판매자에게 자동 노출

> 딜 생성은 무료이며, 포인트 소모 없습니다.

---

## 3. 오퍼 비교 & 선택
딜에 판매자들이 오퍼를 제출하면:
- **가격 여정 맵**에서 가격 변화 추이 확인
- 각 오퍼의 가격, 배송비, 배송 방식, 환불 조건 비교
- 마음에 드는 오퍼 선택 → **예약(참여)**

### 오퍼 확인 포인트
- 총 결제액 = 상품가 + 배송비
- 배송 방식: 택배/직거래/무료배송
- 환불 가능 여부 및 조건
- 판매자 평점, 레벨

---

## 4. 예약 & 결제
1. 오퍼 선택 → **예약하기**
2. 결제 대기 (5분 이내 결제 필요)
3. 결제 완료 → 상태: PAID
4. 판매자가 배송 처리

### 결제 후 흐름
```
PENDING → PAID → (판매자 배송) → SHIPPED → (구매자 도착확인) → CONFIRMED → 정산
```

---

## 5. 배송 추적
경로: /orders → 🚚 배송 조회
택배사 운송장이 등록되면 실시간 배송 상태를 확인할 수 있습니다.
배달 완료 후 3일 내에 수취 확인을 하지 않으면 자동 구매확정됩니다.
배송 상태: 배송준비 → 집하 → 이동중 → 배달중 → 배달완료

> 지원 택배사: CJ대한통운, 한진택배, 로젠택배, 우체국택배, 롯데택배 등

---

## 6. 환불/반품/교환
- 결제 후 배송 전: **전액 환불** 가능
- 배송 후: 상품 문제 시 **분쟁 접수** 후 환불
- 환불 유형: refund(환불) / return(반품) / exchange(교환)

### 환불 유형별 차이
- **구매자 사유**: 배송 시작 후 배송비 환불 불가. PG수수료 구매자 부담.
- **판매자 사유**: 배송비/수수료 판매자 부담. 쿨링 기간 내 배송비 환불 가능.
- **시스템 오류**: 배송비/수수료 시스템 부담.
- **분쟁 결과**: 배송 상태 무관 배송비 항상 환불. 수수료 시스템 부담.

### 환불 신청 방법
1. **내 주문** → 해당 예약 선택
2. **환불 요청** 버튼
3. 사유 선택 및 상세 설명
4. 관리자 검토 후 처리

### 환불 정책 시뮬레이터
다양한 조건(배송 상태, 환불 사유, 수량)을 입력하면
환불 가능 여부, 환불 금액, 배송비 부담, 수수료를 미리 계산할 수 있습니다.
구매자 사유/판매자 사유/시스템 오류/분쟁 결과에 따라 환불 금액이 달라집니다.

---

## 7. 분쟁 해결
상품 미배송, 상품 불량, 설명과 다른 상품 등 문제 발생 시:
1. **내 주문** → 해당 예약 → **분쟁 접수**
2. 사유 입력 (미배송/불량/오배송/기타)
3. 관리자 검토 (최대 14일)
4. 결과 통보 (전액환불/부분환불/환불없음)

> 분쟁 중에는 해당 거래의 정산이 자동 보류됩니다.

---

## 8. 포인트 & 등급
- **결제 시**: +20P 적립
- **환불 시**: -20P 차감
- **포인트 유효기간**: 365일
- **등급**: Bronze → Silver → Gold → Platinum (90일 GMV 기준)

---

## 9. 관전 모드
진행 중인 딜의 가격 흐름을 구경하고 결과를 예측할 수 있습니다.
- 예측이 적중하면 포인트 보상!
- 나중에 비슷한 딜에 참여할 때 참고 데이터로 활용

---

## 10. 자주 묻는 질문
- **Q: 결제 후 취소 가능?** → 배송 전까지 가능, 배송 후는 분쟁 접수
- **Q: 배송이 안 와요** → 분쟁 접수 → 관리자 확인 → 환불 처리
- **Q: 포인트 소멸?** → 적립일로부터 365일 후 소멸
- **Q: 딜 삭제 가능?** → 오퍼가 없으면 삭제 가능, 오퍼 있으면 마감만 가능

## 11. 소셜 로그인
경로: /login
카카오(💬), 네이버(N), 구글(G) 계정으로 간편하게 가입/로그인할 수 있습니다.
소셜 로그인 시 이메일과 닉네임이 자동으로 채워집니다.
기존 이메일로 가입한 계정이 있으면 자동 연동됩니다.

---

## 12. 딜 생성 — 사진으로 찾기
경로: /create-deal (Step 1)
📷 버튼을 눌러 제품 사진을 찍으면 AI가 자동으로 제품을 식별합니다.
최대 3장까지 업로드 가능하며, 가장 정확한 결과로 자동 입력됩니다.
제품 박스나 라벨을 찍으면 더 정확해요.

---

## 13. 딜 생성 — 음성으로 찾기
경로: /create-deal (Step 1)
🎤 버튼을 눌러 "갤럭시 S25 울트라 110만원에 사고 싶어"처럼 말하면
AI가 제품명, 브랜드, 희망 가격, 수량을 자동으로 파싱합니다.
최대 30초까지 녹음 가능합니다.

---

## 14. 딜 생성 — 가격 챌린지
경로: /create-deal (Step 3)
"이 제품이 시장에서 얼마일까요?" — 내 예상 가격을 입력하고 [맞춰보기! 🎯]를 누르세요.
AI가 네이버쇼핑에서 시장가를 조사하고, 근거 URL과 함께 보여줍니다.
액세서리/묶음/중고는 자동으로 제외되며, 제외 사유도 확인할 수 있습니다.
슬라이더로 할인율을 조정하면 목표가가 자동 계산됩니다.

### 시장가 근거 확인
딜 생성 Step 3에서 시장가 옆 [→] 버튼을 누르면 네이버쇼핑 상품 페이지로 이동합니다.
역핑은 부품/액세서리/중고 가격을 걸러내고 본체 가격만 보여줍니다.
