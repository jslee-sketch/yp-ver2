# 구매자 가이드

## 역핑이란?
역핑(YeokPing)은 소비자 주도형 역경매 공동구매 플랫폼입니다.
구매자가 원하는 제품과 가격을 먼저 제시하면, 판매자들이 경쟁 오퍼를 제출합니다.
가장 좋은 조건의 오퍼를 선택하여 구매할 수 있습니다.

### 기존 쇼핑과 차이점
| 기존 쇼핑 | 역핑 |
|-----------|------|
| 판매자가 가격 결정 | 구매자가 희망가 제시 |
| 1:1 거래 | 공동구매로 가격 하락 |
| 가격 비교 어려움 | 오퍼 비교로 최적 선택 |

---

## 회원가입 / 로그인
경로: /register, /login
일반 이메일 가입 또는 소셜 로그인(카카오/네이버/구글)으로 가입 가능합니다.
소셜 로그인 시 역할(구매자/판매자/액추에이터)을 선택하고 닉네임을 설정합니다.
회원가입 완료 후 자동 로그인되어 메인 페이지로 이동합니다.

## 소셜 로그인
카카오(💬), 네이버(N), 구글(G) 계정으로 간편하게 가입/로그인할 수 있습니다.
기존 이메일 계정이 있으면 소셜 계정과 자동 연동됩니다.
마이페이지에서 연동 상태를 확인할 수 있습니다.
신규 소셜 유저는 역할 선택 → 닉네임 설정 → 회원가입 완료 후 자동 로그인됩니다.

## 딜 생성
경로: /deal/create

딜 생성은 무료이며, 포인트 소모 없습니다. 5단계로 진행됩니다.

### Step 1: 제품 찾기
세 가지 방법으로 제품을 검색할 수 있습니다:
- **텍스트 입력**: 제품명을 직접 입력 후 [AI 분석 🔍] 버튼
- **📷 사진으로 찾기**: 제품/박스/라벨 사진을 최대 3장 업로드하면 AI가 자동 인식
- **🎤 음성으로 찾기**: 마이크로 "갤럭시 S25 110만원"처럼 말하면 AI가 제품명/브랜드/가격/수량 자동 파싱 (최대 30초)
검색창에 글자가 입력되면 초록색 테두리와 네온 효과가 나타납니다.
> 자주 묻는 표현: 사진으로 딜, 사진 검색, 이미지 검색, 카메라, 음성으로 딜, 음성 검색, 마이크, 말로 검색, 딜 어떻게 만들어, 딜 만들기, 딜 생성 방법, 딜 생성 단계

### Step 2: 제품 정보 확인
AI가 분석한 브랜드, 카테고리, 제품명, 제품코드, 옵션을 확인하고 수정할 수 있습니다.
유사한 딜방이 이미 있으면 "비슷한 딜방이 있어요!" 카드가 표시됩니다.
기존 딜에 [참여 →] 버튼으로 참여하거나 새로 만들 수 있습니다.
유사 딜 매칭은 키워드 정규화(공백/붙여쓰기 통일)를 통해 "갤럭시S25"와 "갤럭시 S25"도 같은 딜로 인식합니다.
> 자주 묻는 표현: 유사 딜방, 비슷한 딜, 같은 제품 딜, 딜방 참여, 기존 딜 참여

### Step 3: 가격 설정 (가격 챌린지)
"이 제품이 시장에서 얼마일까요?" — 예상 가격을 입력하고 [맞춰보기! 🎯] 버튼을 누릅니다.

#### 3중 소스 가격 합의 엔진
AI가 3개 소스를 교차 검증하여 시장가를 조사합니다:
1. **네이버쇼핑**: 최저가 + 상품 목록
2. **쿠팡**: 쿠팡 파트너스 API 검색 결과
3. **GPT-4o AI 추정**: 제품 특성 기반 시장가 추정

#### 신뢰도 등급
- 🟢 **신뢰도 높음**: 2개 이상 소스가 15% 이내로 일치
- 🟡 **신뢰도 보통**: 소스 간 30% 이내 또는 1개 소스만 확인
- 🔴 **신뢰도 낮음**: 정확한 시장가 확인 불가
- ⚫ **온라인 판매 불가**: 자동차/부동산 등 오프라인 전용 제품

가격 소스 비교 패널을 펼치면 각 소스별 가격, 상품 목록, 건수를 비교할 수 있습니다.
근거 URL을 클릭하면 네이버쇼핑에서 직접 확인할 수 있습니다.

#### 자동 필터링
- 액세서리/케이스/필름/충전기 등 부품 자동 제외
- 세트/묶음/패키지 상품 자동 제외 (검색어 자체가 "세트"면 제외하지 않음)
- 중고/리퍼/반품 상품 자동 제외
- 모델 정확 매칭: "갤럭시 S23" 검색 시 S24/S25/S26 결과 자동 제외
- 브랜드 필터링: 13개 주요 브랜드의 한/영 별칭 자동 매핑 (삼성=Samsung=갤럭시=Galaxy 등)
- 명품(롤렉스/샤넬/에르메스/루이비통/구찌 등)은 ⚠️ 가품/병행수입 주의 경고 표시

슬라이더로 할인율(0~50%)을 조정하면 목표가와 절감액이 실시간 계산됩니다.
> 자주 묻는 표현: 가격 챌린지, 맞춰보기, 시장가, 가격 맞추기, 목표가 슬라이더, 목표가 조정, 시장가 신뢰도, 초록 노랑 빨강, 신뢰도 등급, 신뢰도 차이, 신뢰도 높음 보통 낮음

### Step 4: 기타 요청사항
판매자에게 전달할 추가 요청사항을 입력합니다 (배송 조건, 특별 요청 등).
건너뛰기 가능합니다.

### Step 5: 최종 확인 및 딜 생성
입력한 모든 정보(브랜드, 제품명, 옵션, 시장가격, 목표가격, 수량, 예상 총액)를 확인하고 [딜 만들기] 버튼으로 딜을 생성합니다.

## 딜 탐색
경로: /deals
카테고리별, 키워드별로 딜을 검색할 수 있습니다.
각 딜 카드에는 **딜 번호(#{id})**, 상품명, **목표가(목표 X원)**, 생성일, 상태(LIVE/마감)가 표시됩니다.
딜 카드를 클릭하면 딜 상세 페이지(/deals/{id})로 이동합니다.

딜 상세 페이지에서는 **Deal #{id}** 배지, 오퍼 목록(각 오퍼에 **Offer #{id}** 표시), 가격여정, 타임라인을 확인합니다.

경로: /search
제품명으로 딜을 검색합니다.

## 오퍼 비교 & 선택
딜에 판매자들이 오퍼를 제출하면:
- **가격 여정 맵**에서 가격 변화 추이 확인
- 각 오퍼의 가격, 배송비, 배송 방식, 환불 조건 비교
- 마음에 드는 오퍼 선택 → **예약(참여)**

### 오퍼 확인 포인트
- 총 결제액 = 상품가 + 배송비 (모든 가격은 **N,NNN,NNN원** 형식으로 표시)
- 배송 방식: 택배/직거래/무료배송
- 환불 가능 여부 및 조건
- 판매자 평점, 레벨

## 예약 & 결제
1. 오퍼 선택 → **예약하기**
2. 결제 대기 (5분 이내 결제 필요)
3. 결제 완료 → 상태: PAID
4. 판매자가 배송 처리

### 결제 후 흐름
```
PENDING → PAID → (판매자 배송) → SHIPPED → (구매자 도착확인) → CONFIRMED → 정산
```

## 환불
경로: /my-orders → 해당 예약 → [환불 요청]
배송 전/배송 중/배달 완료에 따라 환불 조건이 다릅니다.
- **구매자 사유**: 배송 시작 후 배송비 환불 불가 (PG수수료는 역핑 플랫폼이 흡수하므로 구매자 부담 아님)
- **판매자 사유**: 배송비/수수료 판매자 부담
- **시스템 오류**: 배송비/수수료 시스템 부담
- **분쟁 결과**: 배송비 항상 환불, 수수료 시스템 부담
> 자주 묻는 표현: 환불 방법, 환불 조건, 배송비 환불, 환불 요청, 취소 방법, 결제 취소

### 환불 유형
- **refund**: 단순 환불 (배송 전)
- **return**: 반품 (배송 후, 상품 회수 필요)
- **exchange**: 교환 (새 상품 재배송)

## 분쟁 신청 (2라운드 AI 중재)
경로: /my-orders → [⚠️ 분쟁 신청]
PAID 또는 SHIPPED 상태의 주문에서 분쟁을 신청할 수 있습니다.
배달 완료 후 7일 이내 분쟁 제기 가능합니다.

### 분쟁 절차 (2라운드)
1. **Round 1**: 분쟁 신청 → 상대방 반론(영업일 3일) → AI 1차 중재 → 양측 수락/거절
   - 양쪽 수락 → 합의 종결 (ACCEPTED)
   - 한쪽이라도 거절 → Round 2 진행
2. **Round 2**: 양측 재반론(영업일 2일) → AI 2차 중재 → 최종 수락/거절
   - 양쪽 수락 → 합의 종결
   - 거절 → 분쟁 종료 + 법적 안내 (소비자보호원 1372, 대한상사중재원 등)

### 증거 첨부
분쟁 신청 시 사진/동영상/텍스트 증거를 첨부할 수 있습니다.
Round 2 재반론 시에도 추가 증거를 제출할 수 있습니다.

### AI 중재
GPT-4o 기반 AI가 양측 주장과 증거를 분석하여 공정한 중재안(환불 유형, 금액)을 제안합니다.
AI 중재 결과는 분쟁 상세 페이지(/disputes/{id})에서 타임라인 형태로 확인 가능합니다.

> 자주 묻는 표현: 분쟁 신청, 분쟁 어디서, 분쟁 접수, 사기 신고, 판매자 사기, AI 중재, 2라운드, 법적 안내

> 분쟁 중에는 해당 거래의 정산이 자동 보류됩니다.
> 영업일 기준 마감 (토/일/공휴일 제외)

## 포인트 & 등급
경로: /points
- **결제 시**: 결제금액의 1% 포인트 적립 (예: 100,000원 결제 → +1,000P)
- **환불 시**: 적립된 포인트 비례 차감 (예: 부분 환불 50%이면 적립 포인트의 50% 차감)
- **추천인 보상**: +20P (추천 코드를 공유하여 친구 가입 시)
- **포인트 사용**: 결제 시 적립 포인트를 사용하여 할인 가능
- **포인트 유효기간**: 365일
- **등급**: Bronze → Silver → Gold → Platinum (90일 GMV 기준, 최소 30만원)
> 자주 묻는 표현: 포인트 사용, 포인트 적립, 추천인, 추천 코드, 등급 업그레이드

## 알림
경로: /notifications
딜 매칭, 오퍼 도착, 배달 완료, 자동 구매확정 등 알림을 확인합니다.
제목/내용 검색, 읽음/안읽음 필터, 채널별 발송 배지(앱/푸시/이메일)를 확인할 수 있습니다.
**푸시 알림(FCM)**: 브라우저 알림을 허용하면 앱을 열지 않아도 중요 알림을 실시간으로 받을 수 있습니다.
> 자주 묻는 표현: 푸시 알림, FCM, 알림 설정, 브라우저 알림

### 알림 설정
경로: /settings/notifications
이벤트별로 앱/푸시/이메일 알림 채널을 ON/OFF 설정할 수 있습니다.
전체 ON/OFF 벌크 버튼으로 한 번에 설정 가능합니다.
22개 구매자 이벤트(딜/오퍼/결제/배송/환불/포인트 등)를 개별 제어합니다.
> 자주 묻는 표현: 알림 설정, 알림 끄기, 푸시 끄기, 이메일 알림

### 관심 상품
경로: /settings/interests
관심 카테고리/제품/모델을 등록하면 관련 딜이 생성될 때 자동 알림을 받습니다.
15개 프리셋 카테고리(스마트폰, 노트북, 태블릿 등) + 직접 입력.
최대 10개까지 등록 가능합니다.
회원가입 시에도 관심 상품을 등록할 수 있습니다 (선택사항).
> 자주 묻는 표현: 관심 상품, 관심 등록, 관심 카테고리, 딜 알림, 매칭 알림

## 실시간 채팅
딜방에서 구매자/판매자 간 **실시간 채팅**이 가능합니다 (WebSocket).
메시지를 보내면 즉시 상대방에게 전달됩니다. 입력 중 표시, 온라인 목록, 접속/퇴장 알림도 지원합니다.
> 자주 묻는 표현: 실시간 채팅, 딜방 채팅, 채팅 방법

## 핑퐁이 (AI 어시스턴트)
화면 우하단 🤖 버튼을 클릭하면 핑퐁이 AI 챗봇이 열립니다.
가격 질문, 환불 정책, 배송 상태, 사용 방법 등 무엇이든 물어보세요.
현재 페이지에 맞는 바로가기 버튼도 제공합니다.
빠른 질문 버튼: "지금 제일 좋은 딜 추천해줘", "내 포인트 얼마야?", "환불 정책이 뭐야?" 등.

## 자주 묻는 질문
- **Q: 결제 후 취소 가능?** → 배송 전까지 가능, 배송 후는 분쟁 접수
- **Q: 배송이 안 와요** → 분쟁 접수 → 관리자 확인 → 환불 처리
- **Q: 포인트 소멸?** → 적립일로부터 365일 후 소멸
- **Q: 딜 삭제 가능?** → 오퍼가 없으면 삭제 가능, 오퍼 있으면 마감만 가능
- **Q: 시장가 신뢰도가 뭐야?** → 3개 소스(네이버+쿠팡+AI) 교차 검증 결과 🟢🟡🔴⚫
- **Q: 사진으로 딜 만들 수 있어?** → Step 1에서 📷 버튼으로 사진 업로드
- **Q: 음성으로 제품 검색?** → Step 1에서 🎤 버튼으로 음성 입력
- **Q: 카카오 로그인 방법?** → 로그인 화면에서 💬 카카오 버튼 클릭
- **Q: 유사 딜방이 뭐야?** → 이미 열린 비슷한 딜이 있으면 참여 안내
