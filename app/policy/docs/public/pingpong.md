# 핑퐁이(Pingpong) 안내 (Public)

## 무엇을 해주나
- 정책/규칙/용어를 자연스럽게 설명
- 번호(예약/오퍼/딜)가 있으면 서버 SSOT로 확정 정보(환불금/배송비/마감)를 안내
- 실시간 외부정보(날씨/뉴스/가격)는 외부조회가 성공했을 때만 사실로 말함

## 알림 관련 FAQ
- **Q: 알림 설정 어디서 해?** → /settings/notifications에서 이벤트별 앱/푸시/이메일 ON/OFF
- **Q: 푸시 알림 어떻게 받아?** → 브라우저 알림 허용 + /settings/notifications에서 푸시 ON
- **Q: 관심 상품 어떻게 등록해?** → /settings/interests에서 프리셋 카테고리 + 직접 입력 (최대 10개)
- **Q: 딜 알림 받고 싶어** → 관심 상품 등록하면 관련 딜 생성 시 자동 알림
- **Q: 알림이 안 와** → /settings/notifications에서 해당 이벤트 앱/푸시 ON 확인
- **Q: 이메일 알림 끄고 싶어** → /settings/notifications에서 이메일 토글 OFF
- **Q: 알림 전체 끄기** → /settings/notifications에서 "전체 OFF" 버튼
> 자주 묻는 표현: 알림 설정, 푸시 알림, 관심 상품, 딜 알림, 알림 안 와, 알림 끄기
